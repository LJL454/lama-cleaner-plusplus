# 写了个图像去水印工具，分享一下

最近需要处理一些带水印的图片，在网上找了一圈工具，要么收费太贵，要么效果不太理想。后来发现了 lama-cleaner 这个项目，思路挺不错的，但用起来有些地方不太顺手，于是就想着自己写一个更适合自己的版本。

## 这个工具能做什么？

就是一个简单的图像修复工具：上传图片，用画笔选中要去掉的区域（比如水印、文字、不需要的物体），AI 会自动填充那块区域。

核心是用了 LaMa 模型，效果在同类型里算是不错的。

## 技术栈

- **后端**：Python + PyTorch
- **前端**：Gradio（比原项目的 Flask 前端轻量一些）
- **模型**：LaMa 为主，也集成了 SDXL 和 Real-ESRGAN
- **其他**：OpenCV 做图像预处理，diffusers 加载扩散模型

## 跟原版有什么不同？

主要是根据自己的需求做了些调整：

1. **前端换成 Gradio**：原项目用 Flask + React，我换成了 Gradio，代码量少很多，维护起来简单
2. **配置管理**：用 dataclass 来组织配置，感觉比原项目的管理方式清晰一点
3. **处理逻辑**：调整了 ROI 检测和边缘融合的策略
4. **加了日志**：方便调试的时候看哪里出了问题

## 使用方式

```bash
pip install -r requirements.txt
python app.py
```

然后浏览器打开 http://localhost:7860 就能用了。

## 实际效果

- 简单背景的水印去除效果还可以
- 复杂背景有时候需要多试几次
- 第一次运行要下载模型文件，大概几百 MB

## 目前的问题

- 部分复杂场景下边缘融合还不够自然，还需要继续调参
- 显存占用还是有点高，低配机器可能跑不动 SDXL

## 适合谁用？

- 不想花钱买商业去水印软件的人
- 对效果要求不是特别高的场景
- 想学习 AI 图像修复的同学（代码结构比较简单，可以看看）
